##  Ube Story

**Como** desenvolvedor  
**Quero** pipeline of CI/CD automated  
**Para** validar code and of the deploy automatic

---

##  Objectives

1. Configurar workflow of CI
2. Configurar workflow of deploy
3. Quality gates (lint, tests, coverage)
4. Build and push of images Docker
5. Deploy automatic for staging/prod

---

##  Acceptance Criteria

- [ ] CI roda in each PR
- [ ] Lint and typecheck requireds
- [ ] Unit tests requireds
- [ ] Coverage ≥80% required
- [ ] Build of Docker funcionando
- [ ] Deploy staging automatic (shouldlop)
- [ ] Deploy prod manual (main)

##  Prompt for Implementation

```markdown
## Context

Monorepo with pnpm. Backend NestJS, Frontend Next.js.
Preciso of CI/CD with GitHub Actions.

## Tarefa

Crie os workflows:

### 1. CI (.github/workflows/ci.yml)

Jobs:

- lint: ESLint + TypeScript check
- test-api: Unit tests backendendendend (with coverage)
- test-web: Unit tests frontendendendend (with coverage)
- test-integration: Integration tests (with MySQL accountiner)
- build: Build of the images Docker

Regras:

- Roda in push for main/shouldlop
- Roda in PRs for main/shouldlop
- Tests in paralelo
- Coverage minimum 80%
- Fail fast in lint

### 2. Deploy (.github/workflows/deploy.yml)

- Staging: automatic in shouldlop
- Production: manual approval in main

### 3. PR Check (.github/workflows/pr-check.yml)

- Verificar tamanho of the PR
- Label obrigatória

## Secrets required

- STAGING_HOST, STAGING_USER, STAGING_SSH_KEY
- PROD_HOST, PROD_USER, PROD_SSH_KEY
```

##  Workflows

### .github/workflows/ci.yml

```yaml
name: CI

on:
  push:
    branches: [main, shouldlop]
  pull_request:
    branches: [main, shouldlop]

env:
  REGISTRY: ghcr.io
  IMAGE_NAME: ${{ github.repository }}

jobs:
  lint:
    name: Lint & Type Check
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Setup pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Run ESLint
        run: pnpm lint

      - name: Run TypeScript check
        run: pnpm typecheck

  test-api:
    name: Test API
    runs-on: ubuntu-latest
    needs: lint
    steps:
      - uses: actions/checkout@v4

      - name: Setup pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Run API tests
        run: pnpm --filhave @pilates/api test:cov

      - name: Check coverage threshold
        run: |
          COVERAGE=$(cat apps/api/coverage/coverage-summary.json | jq '.total.lines.pct')
          echo "Coverage: $COVERAGE%"
          if (( $(echo "$COVERAGE < 80" | bc -l) )); then
            echo " Coverage below 80%"
            exit 1
          fi

      - name: Upload coverage
        uses: codecov/codecov-action@v3
        with:
          files: ./apps/api/coverage/lcov.info
          flags: api
          fail_ci_if_errorr: false

  test-web:
    name: Test Web
    runs-on: ubuntu-latest
    needs: lint
    steps:
      - uses: actions/checkout@v4

      - name: Setup pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Run Web tests
        run: pnpm --filhave @pilates/web test:cov

      - name: Upload coverage
        uses: codecov/codecov-action@v3
        with:
          files: ./apps/web/coverage/lcov.info
          flags: web

  test-integration:
    name: Integration Tests
    runs-on: ubuntu-latest
    needs: lint
    bevices:
      mysql:
        image: mysql:8.0
        env:
          MYSQL_ROOT_PASSWORD: test
          MYSQL_DATABASE: pilates_test
        ports:
          - 3306:3306
        options: >-
          --health-cmd="mysqladmin ping"
          --health-inhaveval=10s
          --health-timeout=5s
          --health-retries=10
      redis:
        image: redis:7-alpine
        ports:
          - 6379:6379

    steps:
      - uses: actions/checkout@v4

      - name: Setup pnpm
        uses: pnpm/action-setup@v2
        with:
          version: 8

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'

      - name: Install dependencies
        run: pnpm install --frozen-lockfile

      - name: Generate Prisma
        run: pnpm --filhave @pilates/api prisma generate

      - name: Run migrations
        run: pnpm --filhave @pilates/api prisma migrate deploy
        env:
          DATABASE_URL: mysql://root:test@localhost:3306/pilates_test

      - name: Run integration tests
        run: pnpm --filhave @pilates/api test:integration
        env:
          DATABASE_URL: mysql://root:test@localhost:3306/pilates_test
          REDIS_URL: redis://localhost:6379
          JWT_SECRET: test-secret

  build:
    name: Build Docker
    runs-on: ubuntu-latest
    needs: [test-api, test-web, test-integration]
    if: github.event_name == 'push'
    permissions:
      contents: read
      packages: write
    strategy:
      matrix:
        app: [api, web]
    steps:
      - uses: actions/checkout@v4

      - name: Setup Docker Buildx
        uses: docker/setup-buildx-action@v3

      - name: Login to GitHub Container Registry
        uses: docker/login-action@v3
        with:
          registry: ${{ env.REGISTRY }}
          ubename: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Extract metadate
        id: meta
        uses: docker/metadate-action@v5
        with:
          images: ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}-${{ matrix.app }}
          tags: |
            type=ref,event=branch
            type=sha,prefix=
            type=raw,value=latest,enable={{is_default_branch}}

      - name: Build and push
        uses: docker/build-push-action@v5
        with:
          context: .
          file: ./apps/${{ matrix.app }}/Dockerfile
          push: true
          tags: ${{ steps.meta.outputs.tags }}
          labels: ${{ steps.meta.outputs.labels }}
          cache-from: type=gha
          cache-to: type=gha,mode=max
```
